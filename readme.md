# 📌 Project Overview

This project predicts the resale price of used cars using Machine Learning.

The application allows users to enter car specifications such as:

- Brand
- Fuel Type
- Seller Type
- Transmission
- Owner
- Manufacturing Year
- Kilometers Driven
- Engine Capacity
- Mileage
- Maximum Power
- Number of Seats

The trained Machine Learning model instantly predicts the estimated resale value of the selected vehicle through an interactive Streamlit web application.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/your-username/Car-Price-Prediction.git
```

## Navigate

```bash
cd Car-Price-Prediction
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run Streamlit

```bash
streamlit run app.py
```
